A/B TESTING and USER EXPERIRNCE OPTIMISATION ANALYSIS FOR DIGITALPATH INNOVATIONS USING TABLEAU

In this project, I conducted a comprehensive A/B test analysis for DigitalPath Innovations, a leading digital marketing solutions company aiming to enhance its website’s user experience and conversion rates. The primary goal was to determine which of two webpage versions (Version A or Version B) performed better across key performance indicators and should implemented as the final design. Using Tableau, I analyzed user behavior data gathered from random users across the two webpage versions to assess engagement metrics, content interaction, referral patterns, and conversion performance. The analysis revealed that while Version B encouraged quicker actions and better video engagement, Version A outperformed in user retention and conversion efficiency. Based on these insights, I recommended a hybrid webpage design that combines the strengths of both versions balancing strong user engagement with high conversion efficiency, drove higher user satisfaction and overall enhanced business performance by 10%.
